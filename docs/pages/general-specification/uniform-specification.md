# 6. 统一规范

## 6.1 代码缩进
&emsp;&emsp;统一使用4个空格进行代码缩进。
```html
<div class="jdc">
    <a href="#"></a>
</div>
```
```ts
let a = 'a';
let b = 'b';
let c = {
    a,
    b,
};
```

## 6.2 注释

### 6.2.1 单行注释  

- 单行注释应写在被注释对象的上方，不要追加在某条语句的后面。注释行的上方需要有一个空行（除非注释行上方是一个块的顶部），以增加可读性。

### 6.2.2 多行注释

- 多行注释不要书写多行的单行注释。

### 6.2.3 文档类注释（函数、类、文件、事件）

- 文档类注释，如函数、类、文件、事件等；都使用 jsdoc 规范。
